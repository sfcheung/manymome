## Notes

### Latent variables

If all variables along a path are latent variables, product term(s) must
be identified by their names because raw scores are not available.

Default uses `"_x_"`. For example, `f1_x_f2` is the product term between
`f1` and `f2`.

### Extracting Point Estimates and Variance-Covariance Matrix

When the point estimates or variance-covariance matrix of the point
estimates are needed, they will be extracted internally using functions
developed for the fit object, which can be a `lavaan`-class object, a
list of the outputs from
[`stats::lm()`](https://rdrr.io/r/stats/lm.html), or a `lavaan.mi`-class
object generated by fitting a model to several datasets using multiple
imputation.
