([Cheung & Cheung, 2024](https://doi.org/10.3758/s13428-023-02224-z))
works to generate bootstrap estimates to be used by other functions to

## Notes

### Extracting Point Estimates

When the point estimates are needed, they will be extracted internally
using functions developed for the fit object, which can be a
`lavaan`-class object, a list of the outputs from
[`stats::lm()`](https://rdrr.io/r/stats/lm.html), or a `lavaan.mi`-class
object generated by fitting a model to several datasets using multiple
imputation.
