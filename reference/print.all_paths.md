# Print 'all_paths' Class Object

Print the content of 'all_paths'-class object, which can be generated by
[`all_indirect_paths()`](https://sfcheung.github.io/manymome/reference/all_indirect_paths.md).

## Usage

``` r
# S3 method for class 'all_paths'
print(x, ...)
```

## Arguments

- x:

  A 'all_paths'-class object.

- ...:

  Optional arguments.

## Value

`x` is returned invisibly. Called for its side effect.

## Details

This function is used to print the paths identified in a readable
format.

## See also

[`all_indirect_paths()`](https://sfcheung.github.io/manymome/reference/all_indirect_paths.md)

## Examples

data(data_serial_parallel)
mod <-
"
m11 ~ x + c1 + c2
m12 ~ m11 + x + c1 + c2
m2 ~ x + c1 + c2
y ~ m12 + m2 + m11 + x + c1 + c2

fit <- sem(mod, data_serial_parallel,
           fixed.x = FALSE)
# All indirect paths
out1 <- all_indirect_paths(fit)
