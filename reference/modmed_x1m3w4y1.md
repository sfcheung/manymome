# Sample Dataset: Moderated Serial Mediation

Generated from a serial mediation model with one predictor, three
mediators, and one outcome variable, with one moderator in each stage.

## Usage

``` r
modmed_x1m3w4y1
```

## Format

A data frame with 200 rows and 11 variables:

- x:

  Predictor. Numeric.

- w1:

  Moderator 1. Numeric.

- w2:

  Moderator 2. Numeric.

- w3:

  Moderator 3. Numeric.

- w4:

  Moderator 4. Numeric.

- m1:

  Mediator 1. Numeric.

- m2:

  Mediator 2. Numeric.

- m3:

  Mediator 3. Numeric.

- y:

  Outcome variable. Numeric.

- gp:

  Three values: "earth", "mars", "venus". String.

- city:

  Four values: "alpha", "beta", "gamma", "sigma". String.
